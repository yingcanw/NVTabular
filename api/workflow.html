Workflow class organizes and runs all the feature engineering and pre-processing operators for your workflow.

  • cont_names (list of str) – Names of the continous columns.

create_final_col_refs()[source]¶

This function creates a reference of all the operators whose produced columns will be available in the final set of columns. First step in creating the final columns list.

create_final_cols()[source]¶

This function creates an entry in the columns context dictionary, not the references to the operators. In this method we detail all operator references with actual column names, and create a list. The entry represents the final columns that should be in finalized dataframe.
